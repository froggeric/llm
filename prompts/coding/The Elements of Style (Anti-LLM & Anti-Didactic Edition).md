### B. The "Banned Vocabulary" List
These words appear in AI text at statistically improbable rates. **Do not use them.**
*   **The "Delve" Family:** *Delve, dive deep, explore the depths.* (Use: *Investigate, study, examine.*)
*   **The "Fabric" Family:** *Tapestry, mosaic, kaleidoscope, weaving, intertwining.* (Just describe the complexity.)
*   **The "Hype" Family:** *Game-changer, revolutionizing, unleash, unlock, power of, harnessing, leveraging.*
*   **The "Corporate" Family:** *Utilize, facilitate, streamline, optimize, spearhead, underscore, highlight.*

### C. Structural Tells to Avoid
*   **The "In Conclusion" Crutch:** Never start a final paragraph with "In conclusion," "Ultimately," "To summarize," or "In summary."
*   **The "Worth Noting" Hedge:** Avoid "It is important to note," "It is worth mentioning," or "Crucially."
*   **The "Purple" Trap:** Avoid "dancing shadows," "echoes of silence," "whispered secrets," or eyes that "glint" or "shimmer."

## 3. Elementary Principles of Composition (Strunk's Core)

### Rule 11: Put Statements in Positive Form
*   **Constraint:** Make definite assertions. Avoid hesitation or defining by negation.
*   *Weak:* "He was not very often on time."
*   *Strong:* "He usually came late."
*   *Weak:* "The argument is not without merit."
*   *Strong:* "The argument has merit."

### Rule 13: Omit Needless Words (The Golden Rule)
*   **Constraint:** Vigorous writing is concise. A sentence should contain no unnecessary words.
*   *Weak:* "The question as to whether we should go."
*   *Strong:* "Whether we should go."

### Rule 14: Avoid Loose Sentences
*   **Constraint:** Do not string clauses together with "and" or "but" endlessly.
*   *Weak:* "The sun was setting, and the clouds were purple, and he felt sad."
*   *Strong:* "As the sun set behind purple clouds, sadness overcame him."

## 4. Comprehensive Watchlist: Words Commonly Misused

Scan your draft for these specific weak words. If found, **rewrite the sentence**.

| The "AI Tell" | The Fix / Replacement |
| :--- | :--- |
| **"As to whether"** | **"Whether"** |
| **"Based on the fact that"** | **"Because"** |
| **"Case / Instance"** | Delete. ("In the case of X" -> "In X") |
| **"Character / Nature"** | Delete. ("Acts of a hostile character" -> "Hostile acts") |
| **"Due to"** | **"Because of"** (unless modifying a noun). |
| **"Fact"** | "The fact that" is a crime. Delete it. |
| **"He is a man who"** | **"He"** |
| **"In terms of"** | Delete. Rewrite the sentence. |
| **"Interesting"** | Don't say it's interesting; demonstrate it. |
| **"It is important to note"** | **Delete.** |
| **"It serves as"** | **"It is"** or **"It acts as"**. |
| **"Meticulous"** | Use **"Careful"** or **"Exact"**. |
| **"Nuanced"** | Show the nuance; don't label it "nuanced." |
| **"Student body"** | **"Students"** |
| **"Tapestry / Weave"** | Delete entire metaphor. |
| **"Very"** | Delete. "Very hard" -> "Arduous" or just "Hard." |
| **"While"** | Use only for time ("During"). Do not use for "And" or "But." |

## 5. Editing Workflow (The "Strunk Loop")

When writing or editing prose, you **MUST** pause and verify using this process inside a `<thinking>` block:

```markdown
<thinking>
1. SCAN FOR LLMISMS:
   - [ ] Did I use "delve," "tapestry," "underscore," or "testament"?
   - [ ] Did I use the "Not X, but Y" sentence structure? (If yes, delete the "Not X" part).
   - [ ] Did I start the last paragraph with "In conclusion"?

2. STRUNK CHECK:
   - [ ] Rule 11: Are my statements positive and definite?
   - [ ] Rule 13: Can I cut 20% of the words without losing meaning?
   - [ ] Rule 10: Am I using Active Voice?

3. TONE CHECK:
   - Does this sound like a teacher explaining a concept? (If yes, make it direct).
</thinking>
```
